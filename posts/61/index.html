往MacBook灌Windows XP

Jun 1st, 2006 | Comments

13英寸宽屏的MacBook，RWTH学生价950欧，我成功怂恿了一位朋友网购，一周多才到货，居然是中国寄过来的，呵呵。

1，装 Windows XP
Windows XP 当然是要装的，虽然 MAC OS X 可以做所有 Windows 能做的事，而且大部分还要做得更好，但毕竟这位朋友原来不是 MAC 的 Fans，所以零时间上手的 Windows XP 还是要装的。


Apple 提供了安装Windows系统工具：Boot Camp。不管是 Apple 的官方网站，还是在装了 Boot Camp 的电脑上都提供了详细的安装说明，这里我就不赘述了。

2，装驱动
装好了Windows，发生的第一件有趣的事就是：取不出光盘。因为MacBook是用的吸入式光驱，弹出键是Keyboard右上方的一个按钮，而初次进入 Windows 是没装这个按钮驱动的。要装驱动必须把光驱中的Windows 安装盘取出来，把驱动盘塞进去。我们经常在光盘上右键，选弹出也可以换光盘，但 Apple 是没有右键的。哈哈，看着那个 Touch Pad 下大大的单键，笑死我了。还好有"茴"字有四样写法：选中光盘，按文件菜单，点 弹出。

Boot Camp 从 Tiger ( MAC OS X )中抽出的驱动不完全。摄像头的就没有，另外那个漂亮的遥控器也没有用。不过瑕不掩瑜，Windows 跑得还是很快的。图中的Cisco Systems VPN Adapter，是RWTH学校VPN上网用的软件虚拟的一张网卡，与 MacBook 无关：


3，鼠标右键
接下来要解决右键的问题，总不能到哪都带个 USB 鼠标吧。观察了一下键盘，发现那个 Apple 键有两个，就像 Windows 键一样。空白键右边那个要是能当鼠标右键的话，还很顺手。改变按键功能不是很难，也就是 Remapping 。有个开源软件就很不错：http://www.autohotkey.com/ 最重要的是这个程序还支持编译成exe文件。这样把编译好的文件丢在启动选项里面，那一进 Windows 就会有右键了。

昨天还用这个软件为电视卡的遥控器编过代码，所以今天这个也就轻车熟路了，可以说相当简单：
#NoEnv
SendMode Input
RWin::RButton

在MacBook 下一试，效果还不错，呵呵。

4，Delete 键
因为朋友在购买的时候选择的是德文键盘，再加上是 Apple 机，所以我找键没那么快，但把我的近视眼找绿了还没发现 Delete 键(德文Entfernen)就有点不正常了。我Google，我Baidu，发现有张图片上MacBook Pro 的英文键盘右边从上往下第二个键就是 Delete ！ MacBook 就是没有！

这下好玩了，没 Delete 键。还好右边的 Apple 键旁边是数字键盘的回车键，那就把它 Remap 吧，加一句在上面的代码后面就好了：
#NoEnv
SendMode Input
RWin::RButton
NumpadEnter::Delete
!q::send @
return
最后那句 !q::send @ 是按德语键盘习惯，把 Alt + q 替换为 @ 。

5，Ghost Windows XP
我最烦的就是装系统了，所以凡是我装好的系统一定要Ghost一份。插入 深山红叶工具盘，打开控制面板，双击启动磁盘：


再选中光盘，点重新启动，记得启动的时候插入一个移动硬盘，Ghost 文件就拷这移动硬盘里面。


Ghost 的过程就不多说了，只要注意选好源盘和镜像保存的盘就没问题了。要备份的分区就是装Windows XP的那个一眼就能看出来的 Fat32 盘。

当然有更简单的调整启动分区的方法：开机的时候按住 传说中的 Option键(就是Alt键)，系统会出现启动提示，如果塞入光盘，还会多加一个光驱选项。这时键盘上的光驱的弹出键是有用的，弹出光盘的时候还有动画，Mac还真注意细节。

在1000欧以下的笔记本中，MacBook 不失为一个相当不错的选择。除显卡略显逊色以外，其它方面都做得不错。小白的优雅气质还是很迷人的。
