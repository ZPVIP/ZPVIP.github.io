清理桌面右键菜单

Jun 1st, 2006 | Comments

安装intel显卡驱动后，在桌面右键菜单多了两项。
Graphics Properties…
Graphics Options
烦之

百度。。。

第一种方法
Win -> Run -> regedit
HKEY_CLASSES_ROOT\Directory\Background\ shellex\ContextMenuHandlers
删除其中的“igfxcui”键

第二种方法
Win -> Run -> 输入 regsvr32 /u igfxpph.dll  回车  
即可去除多余的intel右键菜单;

重新注册就可以加上这个菜单：
Win -> Run -> 输入 regsvr32 igfxpph.dll  回车

Enable Apache Mod_rewrite on Windows

Apr 11th, 2006 | Comments

我用的是xampp的网络编程环境，默认配置下，Apache mod_rewrite 是禁用的。

要开启其实并不难，找到httd.conf文件，把下面代码前面的#去掉就可以了。
重启Apache，用phpinfo()看一下，OK，mod_rewrite装载成功。

#LoadModule rewrite_module modules/mod_rewrite.so
